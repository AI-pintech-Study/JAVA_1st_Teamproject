<h2>4. 기능 명세서</h2>

메인
- 로그인 시 유효성 검사 후 권한에 맞는 접근 제한을 가진 상품 & 회원 관리 이동
- 회원 가입시 필수 입력 항목과 유효성 검사를 거쳐 회원 DB에 저장

상품 관리
- 상품 조회
   -상품 개별 상세 조회
- 상품 추가
- 상품 수정
- 상품 삭제
- 상품 구매


일반 회원 + 관리자 모두 접근 가능 항목
  - 상품 관리
    - 상품 조회
    - 상품 구매
  - 회원 관리
    - 회원 가입
    - 회원 정보 수정
    - 회원 탈퇴

관리자만 접근 가능 항목
  - 상품 관리
    - 상품 추가
    - 상품 수정
    - 상품 삭제
  - 회원 관리
    - 회원 조회
